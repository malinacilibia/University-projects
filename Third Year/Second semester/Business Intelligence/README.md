# Business Intelligence Project — FSEGA

## Overview  
This project was developed as part of the **Business Intelligence** course.  
It focuses on developing an end-to-end data analysis and visualization process in **Tableau**, covering data preparation, transformation, and interactive dashboard design.  
The documentation file included in the project archive provides a full explanation of the objectives, methodology, and results.

---

## How to Open the Project  

1. **Download all three files** from the repository:  
   - `Proiect.twb`  
   - `proiect2.hyper`  
   - `ProiectFlow2.tfl`  

2. **Open Tableau Desktop**.  

3. **Load the data source**:  
   - Double-click `proiect2.hyper` or open it through *Data → New Data Source → Existing Extract*.  

4. **Open the flow**:  
   - Open `ProiectFlow2.tfl` in **Tableau Prep Builder** to visualize the data cleaning and transformation steps.  

5. **Open the main workbook**:  
   - Open `Proiect.twb` in Tableau Desktop.  
   - If prompted, reconnect to the `.hyper` file using the same folder path.  

6. Once loaded, explore the **dashboards and worksheets** to view KPIs, charts, and analytical results.
