# Task Management System — Node.js Web Application

## Overview
This project represents a full-stack web application developed using **Node.js**, designed to efficiently manage and track daily tasks.  
The platform allows users to add, edit, delete, and mark tasks as complete while organizing them by **priority** and **status**.  
It provides a minimal and elegant interface for productivity tracking and team collaboration.  
The application was developed as part of the *Design and Development of Websites and Web Portals* course.

---

## Application Features

### 1. Dashboard — Tasks Overview
The main interface displays all existing tasks in a structured table, allowing users to **filter, search, and mark tasks as complete**.  
It includes color-coded priorities and real-time updates for completed and in-progress activities.

![Dashboard](../screenshots/nodejs%20(3).png)

---

### 2. Add New Task
Users can easily add new tasks by providing a title, description, and priority level.  
All data is validated before saving, ensuring data consistency.

![Add Task](../screenshots/nodejs%20(1).png)

---

### 3. Completed Tasks
A separate view lists all the completed tasks with options to **delete** or review them.  
This helps maintain a clear separation between ongoing and finalized work.

![Completed Tasks](../screenshots/nodejs%20(4).png)
